# 🎯 THIX ID - Identidade Digital Segura

Uma plataforma completa de identidade digital segura com múltiplos serviços integrados.

## 🌐 Acesso da Aplicação Web

**URL de Produção:** https://depot-data-thix-central.github.io/00/

## ✨ Recursos

- **🔐 Autenticação Segura** com Supabase
- **🏥 THIX Santé** - Sistema de saúde completo
- **💰 THIX Money** - Pagamentos e transações
- **🛍️ THIX Market** - Marketplace
- **📱 Suporte Multiplataforma** - Web, iOS, Android
- **💬 Chat em Tempo Real** - Comunicação integrada
- **📚 Educação e Treinamento** - Plataforma de aprendizagem
